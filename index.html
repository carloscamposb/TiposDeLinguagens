Linguagens Tipagem Organização Usabilidade Vantagens Desvantagens Minha opnião

JavaScript

Tipagem dinâmica (Pode ser inferida pela linguagem sem ser escrita de forma explicita o tipo de dado) e fraca (Conversões automáticas entre tipos diferentes de dados)

Programação estruturada

Muito usado para aplicações web

Maior capacidade de suportar todos os navegadores modernos e produzir um resultado equivalente.

É vulnerável a brechas de segurança; Pode ser usada para executar programas maliciosos sem que o usuário saiba.

Ela é muito versátil podendo ser usada no back-end também. Além disso tem um grande número de recursos

Java

Tipagem estática (existe um sistema de tipos bem definido e que é validado durante o tempo de compilação do código) e forte (Verificações de variáveis são rigorosas a execução do código)

Programação Orientada a Objeto

Criação de aplicações complexas e dinâmicas principalmente no meio empresarial

Linguagem muito segura com alto desempenho

É uma linguagem muito prolixa,maior complexidade e tempo para gerar códigos

Java dentre todas as linguagens mostra uma maior robustez e confiabilidade. Isso é extremamente positivo

PHP

Tipagem dinâmica (Pode ser inferida pela linguagem sem ser escrita de forma explicita o tipo de dado) e fraca (Conversões automáticas entre tipos diferentes de dados)

Originalmente foi criado para ser uma linguagem de programação estruturada, porém após a versão 5, o suporte à orientação a objetos foi implementada e passou a ser muito utilizada.

Sistemas operacionais (Linux, Unix, Microsoft Windows, macOS e outros). Também tem suporte à maioria dos servidores web.

É uma linguagem de código aberto, o que o torna mais flexível e personalizável e tem boa integração com banco de dados (ex.: MySQL)

Não há uma padronização entre as suas versões, nem todos os comandos funcionam da mesma forma.

Essa linguagem tem uma vasta comunidade para tirar dúvidas então é aparentemente fácil de aprender. E demostra ser muito boa na construção de sistemas operacionais devido ao seu dinamismo

Python

Tipagem dinâmica (Pode ser inferida pela linguagem sem ser escrita de forma explicita o tipo de dado) e forte (Verificações de variáveis são rigorosas a execução do código)

Programação orientada objeto

Aplicações da Web, desenvolvimento de software, ciência de dados e machine learning (ML)

Linguagem simples e intuitiva, com uma sintaxe fácil de entender. Alta aplicabilidade

Desempenho mais lento em comparação com outras linguagens para análise de dados.

A facilidade do python atrai muitos programadores e uma infinidade de coisas podem ser construidas por meio dessa linguagem

C#

Tipagem estática (existe um sistema de tipos bem definido e que é validado durante o tempo de compilação do código) e forte (Verificações de variáveis são rigorosas a execução do código)

Programação Orientada a objeto

Muito utilizada para jogos desenvolvidos para Windows e Xbox

Possui integração com Windows. Possui código compilado em linguagem padrão, independentemente do Sistema Operacional e arquitetura de destino.

Necessidade de compilar a cada vez que o código for alterado.

A linguagem C# em alguns pontos se assemelha ao JAVA. Apesar de não ser flexivel tem uma aplicabilidade grande na empresa de jogos e isso chama muito a atenção dos programadores

Ruby

Tipagem dinãmica (Pode ser inferida pela linguagem sem ser escrita de forma explicita o tipo de dado) mas também oferece suporte a tipagem estática e forte.

Programação orientada a objetos

Muito usado para aplicações web

Linguagem de programação muito concisa, o que torna o desenvolvimento de software mais rápido e eficiente.

Linguagem relativamente nova, o que significa que pode não ter a mesma disponibilidade de recursos que linguagens mais estabelecidas, como Java.

Mesmo sendo uma linguagem nova a Ruby parece ser uma linguagem bastante segura, versátil e com alta aplicabilidade para diversos projetos web
